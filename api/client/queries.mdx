## Overview

Query functions fetch data from the local SQLite database. All queries are type-safe and follow a consistent pattern.

## Query Pattern

All queries follow this structure:

```typescript
const result = await mediator.executeQuery({
  type: 'query.name',
  accountId: string,
  workspaceId: string,
  // ... query-specific parameters
});
```

### node.search

Search nodes by text content.

<ParamField path="type" type="'node.search'" required>
  Query type identifier
</ParamField>

<ParamField path="searchQuery" type="string" required>
  Search text
</ParamField>

<ParamField path="accountId" type="string" required>
  Account ID
</ParamField>

<ParamField path="workspaceId" type="string" required>
  Workspace ID
</ParamField>

<ParamField path="types" type="string[]" optional>
  Filter by node types (e.g., ['page', 'database'])
</ParamField>

<ParamField path="limit" type="number" optional>
  Maximum number of results
</ParamField>

<ParamField path="offset" type="number" optional>
  Pagination offset
</ParamField>

<ParamField path="titleOnly" type="boolean" optional>
  Search only in titles, not content
</ParamField>

<ParamField path="dateRange" type="'today' | 'week' | 'month'" optional>
  Filter by updated date
</ParamField>

<ParamField path="spaceId" type="string" optional>
  Filter by space (root_id)
</ParamField>

<ResponseField name="output" type="NodeSearchResult[]">
  Array of search results with snippets and ranking
</ResponseField>

```typescript
const results = await mediator.executeQuery({
  type: 'node.search',
  searchQuery: 'project documentation',
  accountId: 'account-123',
  workspaceId: 'workspace-456',
  types: ['page', 'database'],
  limit: 20,
  titleOnly: false,
  dateRange: 'week'
});

results.forEach(result => {
  console.log(`${result.nameSnippet} (rank: ${result.rank})`);
  console.log(`Content: ${result.contentSnippet}`);
});
```
